Formatação com HTML5 samântico e não semântico

Negrito

Texto em negrito pode ser aplicado de duas formas, com a tag <b> ou a tag <strong>

A diferença entre elas é o seu valor semântico para os macânismos de busca, a tag <b> é uma tag de negrito visual, apenas o usuário percebe, enquanto que com a tag <strong> nós temos uma tag semântica para destacar um trecho do site em negrito.

Itálico

Texto em itálico pode ser aplicado de duas formas, com a tag <i> ou com a tag <em>

A diferença entre elas é o seu valor semântico para os mecânismos de busca, a tag <i> é uma tag de ênfase visual do texto, apenas o usuário percebe, enquanto que com a tag <em> nós temos uma tag semântica para ênfase de trechos de texto no nosso site.

Sublinhado

Para sublinhar o texto com um destaque textual ou algo do tipo, utilizamos a tag <ins>, uma tag semântica para destacar partes do texto em sublinhado,informaçôes adicionais e outros afins.

Marca texto web

Para adicionar uma marcação Como essa aqui no seu codigo, é necessário utilizar a tag <mark>

Demais formas de edição textual

Utilizando a tag <small>

Texto um pouco menor que os demais em alguns trechos | a partir daqui o trecho deve estar menor que o anterior

Utilizamos a tag <b=small>

Texto com um risco em cima, para simbolizar que os mesmo foi removido ou será removido | a partir daqui o trecho deve ser removido do codigo em breve

Utiliizamos a tag <sub>

Texto com letras em um nivel a baixo do normal, muito utilizado em fórmulas químicas | C21H3002

Utilizando a tag <sup>

Texto com letras em um nível acima do noromal, muito utilizado em fórmulas matemáticas | X2
